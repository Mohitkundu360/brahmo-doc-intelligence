# Data Sources

## Test Contracts

All test contracts in `test-documents/` were **generated specifically for this assessment** using AI tools (Claude). They are synthetic documents created to match the documented differences in the Setup Guide. They do not represent real legal agreements or real parties.

### NDA v1 / NDA v2
- **Source**: AI-generated (Claude Sonnet) based on standard NDA structure
- **Purpose**: Test clause comparison with 4 known differences (liability cap, term, non-solicitation, dispute resolution)
- **Jurisdiction**: Indian law (Delhi courts / SIAC arbitration scenario)

## Firm Knowledge Nodes

10 knowledge nodes (C-010 through D-012) sourced directly from the Assessment Setup Guide provided by BRAHMO/Astroum AI. These represent fictional firm policies and past case decisions created for the assessment.
